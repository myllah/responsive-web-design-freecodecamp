Variables

In Python, you don't need to declare variables before using them. A variable is created the moment you first assign a value to it.

x = 3 y = "Banana" print('x:', x) print('y:', y)

We don't need to declare the varibles's type. Also, we are able to change the type, even after we already gave a value.

x = 3           # In this moment, the type of x is int x = "Banana"     # And now we just changed the type of x to string print(x)
Data Types

We can get the type of a variable with the type() function. This function returns the class type of the argument passed as parameter.

x = 3 y = "Banana" print("type of x:", type(x)) print("type of y:", type(y))

We are able to specify the data type of a variable. This is called casting

x = str(32) y = int(32) z = float(32) print('x:', x)   # x is '32' print('y:', y)   # y is 32 print('z:', z)   # z is 32.0

Python has five standard data types:

  • String

    Strings in Python stores a set of characters, and we are able to access parts of the string:

    x = "Hello World" print(type(x)) print(x)   # Prints the string x print(x[:5])   # Prints the string x until charactere 5 print(x*2)   # Prints the string x twice

  • Numbers(int, float, complex)

    Number data types store numeric values:

    num_int = 3 print(type(num_int)) num_float = 5.0 print(type(num_float)) num_complex = 2 + 3j print(type(num_complex))

  • List

    A list is a compound data type where you can group values together. Lists are very similar to arrays in C. One difference between them is that all the items belonging to a list can be of different data type.

    list = [ 'Banana', 'Grape', 'Apple', 'Orange' ] print(type(list)) print(list)   # Prints the list print(list[0])   # Prints first element of the list print(list[2:])   # Prints elements starting from 3rd element print(list*2)   # Prints the list twice

  • Tuple

    A tuple is a data type very similar to the list data type. As differences, we can notice that the tuples are enclosed in parenthsis, while the lists are enclosed in brackets. A tuple consists of a number of values separated by commas. Unlike lists, tuples does not support item assignment, meaning that they cannot be updated, that is why we considere the tuples as read-only lists

    tuple = ('Banana', 'Grape', 'Apple', 'Orange') print(type(tuple)) print(tuple)   # Prints the list print(tuple[0])   # Prints first element of the tuple print(tuple[2:])   # Prints elements starting from 3rd element

  • Dictionary

    Python's dictionaries are kind of hash table type:

    dict = {"name" : "Myllah", "age" : 20, "pets": ["Kyan", "Kaya"]} print(dict)   #Prints the dict print(dict['name'])   #Prints value for "name" key print(dict['pets'])   #Prints value for "pets" key print(dict.keys())   # Prints all the keys print(dict.values())   # Prints all the values
Reserved Words

Python reserves a set of keywords that designates special language functionalities. These are reserved words, therefore we can't use them as a variable name, function name or any other identifier.

We can use the code below to see the reserved words:

import keyword print(keyword.kwlist)

These were the returned words:
False, None, True, and, as, assert, break, class, continue, def, del, elif, else, except, finally, for, from, global, if, import, in, is, lambda, nonlocal, not, or, pass, raise, return, try, while, with, yield

Indentation

Python does't provide braces to indicate blocks of code. We use indentation, and all statements with the same amount of of whitespace belong to the same block of code.

For example:

num = 1 if num == 1:   print("Number is 1") else:   print("Number is NOT 1")
Comments

All characters after the # are comments and the Python interpreter ignores them.

num = 99 #This is a comment if num == 1: #This is a comment   print("Number is 1")   #This is a comment #This is also a comment ;) else:   print("Number isNOT 1")
Conditions

Python uses boolean logic to evaluate expressions producing True or False as output.

We can use the if by itself:

num = 3 if ( num == 3 ) : print("Num is 3") print("Good bye!")

We can use the if...else:

a = 5 b = 99 if b > a:   print("b is greater than a") else:   print("b is not greater than a")

Or we can use the nested statments if...elif...else:

num = 0 if num == 1:   print("Number is 1") elif num == 0:   print('Number is 0') else:   print("Number is NOT 0 and is NOT 1")
Loops

The lines of code are executed sequentially, but many situations requests to repeat the same block of code repetitively.

For

The For loop iterates to each item in the list, until the end of the sequence:

fruits = ['banana', 'grape', 'mango', 'orange'] for fruit in fruits:   print('Current fruit :', fruit)

While

The While loop iterates while the condition is true. The moment the condition becomes false, the loop stops, and it's executed the line immediately after the loop.

i = 10 while i >= 1:   print(i)   i -= 1
